Websocket
============
## ws.SubscribeBlock

订阅区块

**Parameters**
N/A

**Example**

```json
{
  "jsonrpc": "2.0", 
  "id": "1",
  "method": "ws.SubscribeBlock"
}
```

**Returns**

1. `result` bool - 成功/失败

**Example**

```json
{
  "jsonrpc":"2.0",
  "id":"1",
  "result":true,
  "error":{
    "code":0,
    "message":"ok"
    }
}
```
