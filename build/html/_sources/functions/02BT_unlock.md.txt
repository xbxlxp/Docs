## BT Token Unlock合约

BT Token_unlock合约是控制Bithumb Chain公链基本价值载体BT Token的解锁的合约。BT Token的总发行量为5亿。

合约地址：XagqqFetxiDb9wbartKDrXgnqLahHSe2VE

### 合约方法

1、**解锁BT Token**

* 方法名称：unlock
* 说明：该方法只能由定时器合约调用，定时器合约会每日调用一次该方法，从而实现BT Token的解锁。解锁的总量分别发放给dex和staking合约
* 传入参数及意义：
  * 无
* 返回值及其意义：
  * 无

* 触发event：
  * unlock

```json
    {
      "name":"unlock",
      "inputs":[
        {
          "name":"from",
          "type":"account"
        },
        {
          "name":"to",
          "type":"account"
        },
        {
          "name":"amount",
          "type":"uint64"
        }
      ]
    }
```

2、**提取所有解锁的余额**

* 方法名称：withdrawAll
* 传入参数及其意义：
  * 传入一个参数account类型，代表账户
* 返回值及其意义
  * 提取总量

* 触发event：
  - 无
