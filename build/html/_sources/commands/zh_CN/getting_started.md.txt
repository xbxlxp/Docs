# 快速开始

## 概述

星际价值网络客户端 `xtar CLI` 是一个用 `Go` 语言编写的命令行应用程序，其主要功能如下：

- 运行和管理全节点
- 管理帐户密码
- 管理原生资产及发行Token
- 管理区块数据
- 显示区块链信息
- 构建、签署和发送交易



## 安装

### [下载安装包](https://dev-docs.ont.io/#/docs-cn/xtar-cli/01-install?id=下载安装包)

基于 Go 语言，我们提供了通用的星际价值网络客户端安装程序，你可以通过下列两种方式快速获取：

- 点击 [这里](https://github.com/xtario/xtar/releases) 下载星际价值网络客户端安装程序。

- 通过 `curl` 命令获取最新的客户端。

  ```shell
  curl https://dev.xtar.io/xtar_install | sh
  ```

### [使用源码编译](https://dev-docs.ont.io/#/docs-cn/xtar-cli/01-install?id=使用源码编译)

此外，你也可以从源码编译本体客户端。

1. 获取源码：

   ```shell
   go get github.com/xtario/xtar
   ```

2. 使用包管理工具 `glide` 获取依赖：

   ```shell
   cd $GOPATH/src/github.com/xtario/xtar
   glide install
   ```

3. 编译源码：

   ```shell
   make all
   ```

源码成功编译后会生成两个可执行程序：

- xtar-cli：星际价值网络客户端。



## 快速开始

当你安装好xtar的客户端后，便可以使用以下命令进行一些常用操作。

**查看帮助信息**

```shell
xtar-cli help
```

**连接到主网**

```shell
xtar-cli
```

或者

```shell
xtar-cli 1
```

**连接到测试网**

```shell
xtar-cli 2
```

**启动测试模式**

在测试模式下，节点独立运行，默认的 `资源` 为 0。

```shell
xtar-cli 0
```

或者

```shell
xtar-cli --testnet
```

第一次启动测试模式之前，需要创建钱包文件。

```shell
xtar-cli account new
```
