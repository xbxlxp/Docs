Token合约¶

token合约是Bithumb Chain发行token和管理token合约。token合约对注册发行的进行token生命周期的管理：注册token、发行token、销毁token、变更token发行人、接受成为新的token发行人、查询token发行人、查询token符号、查询token精度、查询token总发行量、token转账、查询token余额、授权token转账、授权token转账提现、查询token授权提现剩余额度的功能。

合约地址：XagqqFetxiDb9wbartKDrXgnqLahUovwfs

可调用合约方法¶

1、注册token

  • 方法名称：register

  • 描述：注册登记发行token的信息，注册的token将存入到issuer发行人的地址中

  • 传入参数及其意义：

    • 传入一个ContractArg合约参数

      • 1.是一个结构体struct代表注册登记token的信息，该结构体含有9个成员：

          1. symbol指定token的象征符号。

          1. decimal指定 token 的小数位。

          1. totalSupply指定 token 的发行总量

          1. issuer指定 token 的发行人地址

          1. name指定token合约名称

          1. author指定 token 的合约所有者

          1. version指定token合约的版本

          1. email指定token合约的联系邮箱

          1. description指定token合约的描述信息

{
  "name":"tokenInfo",
  "type":"struct",
  "components":[
    {
      "name":"symbol",
      "type":"string"
    },
    {
      "name":"decimal",
      "type":"uint8"
    },
    {
      "name":"totalSupply",
      "type":"uint64"
    },
    {
      "name":"issuer",
      "type":"account"
    },
    {
      "name":"name",
      "type":"string"
    },
    {
      "name":"author",
      "type":"string"
    },
    {
      "name":"version",
      "type":"string"
    },
    {
      "name":"email",
      "type":"string"
    },
    {
      "name":"description",
      "type":"string"
    }
  ]
}

  • 返回值及其意义：

    • 无

  • 触发event：

    • register

2、发行token

  • 方法名称：issue

  • 描述：将登记注册的token发行到指定的账户

  • 传入参数及意义：

    • 传入三个ContractArg合约参数

      • tokenAddress指定token合约地址:

      • amount指定token发行的数量

      • source指定token的发行者地址或者销毁token的地址

{
  "name":"tokenAddress",
  "type":"account"
},
{
  "name":"amount",
  "type":"uint64"
},
{
  "name":"source",
  "type":"string"
}

  • 返回值及其意义：

    • 无

  • 触发event：

    • issue

可触发合约事件event¶

1.token注册事件register

  • event名称：register

  • 描述：记录token注册记录的收据

  • event_logs参数及其意义：

    • 第一个参数为event名称

      • 1.register

    • 第二个参数开始代表token注册后相关实体:

      • 2.token合约地址

      • 3.token符号

      • 4.token精度

      • 5.token发行总量

{
  "name":"symbol",
  "type":"string"
},
{
  "name":"decimal",
  "type":"string"
},
{
  "name":"totalSupply",
  "type":"uint64"
},
{
  "name":"tokenAddress",
  "type":"account"
}
